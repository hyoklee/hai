# hai

H Artificial Intelligence

## Simplify Your AI Workflow and Data Management

HAI is an ideal solution for managing all your AI data needs -
from raw training data to model data.

### Query HAI in Natural Langauge / Image / Sound / Video

  * Upload HAI file(s) as source for LLM.
  * Query HAI contents in Natural Langauge.
  * Manipulate data. (e.g., Create image from sound)
  * Find a similar dataset given the image, sound, and video.

### Make Bigdata AI-ready using HAI

 * Save metadata with data.
   * algorithms and their versions used, model parameters, authors, etc.  
 * Save prompts / training / model / testing data in HAI.
 * Save knowledge graph (semantic network) in HAI.

### Reduce Bigdata Storage using HAI

 Filtering Bigdata with AI is a solution to reduce the burden of managing
a large amount of training, testing, and model data.

  HAI can automatically sanitize your data in a scalable manner
  when you archive data in HAI.
  It can save a lot of space by storing only models, not real data.
  
  HAI can 
  * store everything including algorithms to use and learned models.
  * link to the raw data for provenance.
  * set a time to remove raw data and a desired accuracy threshold to
    prune models and data.
  * run several ML algorithms in parallel.
